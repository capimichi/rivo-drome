# Rivo Drome

**Rivo** (dal latino *rivus*, "ruscelletto", flusso continuo) + **Drome** (da Navidrome).

Proxy per [Navidrome](https://www.navidrome.org/) che genera canzoni fresche on the fly, trasformando lo stream musicale in un flusso sempre nuovo.

## Idea

Rivo Drome si inserisce come middleware tra il client e Navidrome, intercettando le richieste e arricchendo la libreria con brani generati al volo, remix, o contenuti freschi da fonti esterne — il tutto in modo trasparente per l'utente finale.
